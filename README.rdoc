== bédof

Cette application consolide et crawl le web pour constituer une base de données (BDD)
avec adresses emails des organismes de formation (OF) en France.

BEDOF travaille avec 4 sources de données :

* La liste officielle des organismes de formation (listeof.travail.gouv.fr), crawlée par l'api "OF" de Kimonolabs

* Societe.com, crawlé par import.io

* BING, crawlé par bédof

* QuickEmailVerification, appelé par bédof
